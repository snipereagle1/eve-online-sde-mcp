# Task: Startup Scan & SdeStore

## Description

Implement linear startup scan of all JSONL files to build in-memory byte-offset indexes. Produce `SdeStore` with all `SdeIndex` fields plus derived maps.

## Acceptance Criteria

- [ ] `SdeIndex` struct: `path: PathBuf`, `id_index: HashMap<u64,u64>`, `name_index: HashMap<String,u64>`
- [ ] `SdeStore` struct with all 16 indexes per PRD §10.2 plus `product_to_blueprint` and `stargate_graph`
- [ ] Scan reads each JSONL file line by line, records byte offset before each line
- [ ] Key extraction: minimal JSON parse — find `"id":` or first key value (no full deserialize)
- [ ] Name extraction: find `"name":{"en":"..."` pattern (regex or manual scan)
- [ ] Name stored lowercased in `name_index`
- [ ] Blueprints scan: extract `activities.manufacturing.products[0].typeID` → `product_to_blueprint` map
- [ ] Stargates scan: extract `systemID` + `destination.systemID` → `stargate_graph` adjacency list (bidirectional)
- [ ] `indicatif` progress bar to stderr (files scanned / total)
- [ ] Scan < 10s on full SDE (benchmark or timing log at debug level)
- [ ] `SdeStore` wrapped in `Arc` for sharing across async tool handlers
