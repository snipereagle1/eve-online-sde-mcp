# Task: Release Workflow

## Description

Create `.github/workflows/release.yml` that cross-compiles for 3 platforms on `v*` tag push, assembles an MCPB bundle, and creates a GitHub Release.

## Acceptance Criteria

- [ ] `.github/workflows/release.yml` created
- [ ] Trigger: push of `v*` tag
- [ ] Parallel matrix: `x86_64-unknown-linux-gnu`, `x86_64-pc-windows-msvc`, `aarch64-apple-darwin`
- [ ] macOS ARM built on `macos-latest` runner (avoids cross-compilation complexity)
- [ ] Linux and Windows built on `ubuntu-latest` using `cross` or native toolchain
- [ ] MCPB assembly step: collect 3 binaries → `server/` dir → attach `manifest.json` (from task 17) → `mcpb pack`
- [ ] GitHub Release: attach 3 platform binaries + `eve-sde-mcp.mcpb`, auto-generated changelog from git log
- [ ] `mcpb` installed in CI via release artifact download if not in PATH
