## Technical Details

```rust
fn fetch_by_id(index: &SdeIndex, id: u64) -> anyhow::Result<serde_json::Value> {
    let offset = index.id_index.get(&id)
        .ok_or_else(|| anyhow::anyhow!("ID {} not found", id))?;
    let mut file = File::open(&index.path)?;
    file.seek(SeekFrom::Start(*offset))?;
    let mut line = String::new();
    BufReader::new(file).read_line(&mut line)?;
    Ok(serde_json::from_str(&line)?)
}
```

Language filter: after deserializing response, if `SDE_LANGUAGE` is set, walk name/description fields and keep only matching language key (or leave as-is if not a localized field).

Tool registration: rmcp uses a router/handler pattern. Each tool gets name, description, input schema (JSON Schema), and async handler fn.
