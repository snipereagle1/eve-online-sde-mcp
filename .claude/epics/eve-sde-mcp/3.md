# Task: Storage & Configuration

## Description

## Acceptance Criteria

- [ ] `Config` struct via `clap`: `--data-dir`, `--log-level`, `--redownload`
- [ ] Env var fallbacks: `SDE_DATA_DIR`, `SDE_LANGUAGE`, `RUST_LOG`
- [ ] Data dir resolves: `SDE_DATA_DIR` → `$XDG_DATA_HOME/eve-sde-mcp` → `~/.local/share/eve-sde-mcp` (Linux/macOS), `%APPDATA%\eve-sde-mcp` (Windows)
- [ ] `meta.json` read/write: `{ "build": u64, "release_date": String, "etag": String }`
