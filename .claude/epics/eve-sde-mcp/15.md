# Task: CI Workflow

## Description

Create `.github/workflows/ci.yml` that runs fmt, clippy, test, and release build on every push and PR to `main`.

## Acceptance Criteria

- [ ] `.github/workflows/ci.yml` created
- [ ] Triggers: `push` and `pull_request` on `main`
- [ ] Steps: `cargo fmt --check`, `cargo clippy -- -D warnings`, `cargo test`, `cargo build --release`
- [ ] `PINNED_BUILD` env var set to `3333874` for test step — no network calls
- [ ] Uses fixture data (`SDE_DATA_DIR` pointed at `tests/fixtures/sde/`)
- [ ] Runs on `ubuntu-latest`
- [ ] Caches `~/.cargo` and `target/` for speed
